Agenda de actividades y eventos

Group01: Aïcha Cheridi, Alexandre Morel, Adrien Simard 

R1. Data are in the smart cities domain.
This dataset is about activities of different types that are going to be held or that are being held, in municipal centers or associated with the activity managed by the City Council. You can find the activities of Cultural Centers, Sociocultural, Youth Centers, Libraries, Senior Centers, Museums, District Boards, etc.

R2. Data are available as a CSV file.

Yes, the dataset is available as a CSV file : https://datos.madrid.es/portal/site/egob/menuitem.c05c1f754a33a9fbe4b2e4b284f1a5a0/?vgnextoid=57be24206a91b510VgnVCM2000001f4a900aRCRD&vgnextchannel=374512b9ace9f310VgnVCM100000171f5a0aRCRD&vgnextfmt=default

 

R3.  Data have an open license so they can be published.

The general conditions allow the reuse of documents for commercial and non-commercial purposes. Reuse means the use of documents held by the Madrid City Council, provided that such use does not constitute a public administrative activity.

R4. Data can be easily linked with generic real-world entities (e.g., locations).

There are several ways to link the data with generic real-world entities. For example with these attributes : Distrito, Barrio, Código Postal, Latitud, Longitud…

R5. Documentation exists for the data (optional).

Yes, documentation exists : https://datos.madrid.es/FWProjects/egob/Catalogo/XComun/Ficheros/Estructura_DS_eventos_wwwmadrid.pdf

R6: Data can come from multiple data sources (optional).

Data come from the municipal website www.madrid.es.
